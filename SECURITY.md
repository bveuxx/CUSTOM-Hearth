# Security Policy

## Supported versions

Hearth is an actively developed Obsidian plugin. Security fixes target the
**latest released version** — please update to it before reporting an issue.

## Reporting a vulnerability

If you believe you've found a security issue in Hearth, please report it
privately rather than opening a public issue.

- Preferred: use GitHub's
  [**Report a vulnerability**](https://github.com/ondreu/Hearth/security/advisories/new)
  flow (Security → Advisories) so the report stays private until a fix is out.

Please include what the issue is, how to reproduce it, and its potential impact.
I'll acknowledge your report as soon as I can and keep you updated on the fix.

Hearth runs entirely inside Obsidian on your own device and doesn't operate any
server or collect data, so the realistic surface is things like unsafe handling
of vault content or embedded web pages — reports in that vein are especially
welcome.
